# an NES emulator

a work-in-progress NES emulator written for Windows

## usage

`nes.exe [ROM file]`

## features

- cycle accurate CPU and PPU emulation
- support for NROM games (tbd)

## keys

NONE yet (joypad yet to be implemented)
