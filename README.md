## usage

`nes.exe [ROM file]`

## features

- cycle accurate CPU and PPU emulation
- support for NROM, MMC1 games (tbd)

## keys

```
-----------------------+
|                +---+ |
|   ↑        +---| s | |
| ← . →      | a |---+ |
|   ↓        +---+     |
|    lshift    enter   |
-----------------------+
```

(arrows are arrow keys)
